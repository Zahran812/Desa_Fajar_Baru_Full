## üéØ LANGKAH SELANJUTNYA (Yang Harus Dilakukan)

### **MINGGU 1: Setup Google (PRIORITAS TERTINGGI!)** ‚≠ê‚≠ê‚≠ê

#### **1. Google Search Console** (15 menit)
```
1. Buka: https://search.google.com/search-console
2. Login dengan Google account
3. Klik "Add property"
4. Masukkan: https://desafajarbaru.id
5. Verifikasi ownership (HTML tag method)
6. Submit sitemap: https://desafajarbaru.id/sitemap.xml
7. Request indexing untuk homepage
```

**WAJIB! Tanpa ini website tidak akan muncul di Google!**

#### **2. Google Business Profile** (20 menit)
```
1. Buka: https://business.google.com
2. Cari "Desa Fajar Baru" atau "Pemerintah Desa Fajar Baru"
3. Klaim bisnis (jika ada) atau buat baru
4. Isi data lengkap:
   - Nama: Pemerintah Desa Fajar Baru
   - Kategori: Government Office
   - Alamat lengkap
   - Nomor telepon
   - Website: https://desafajarbaru.id
   - Jam operasional
   - Deskripsi (gunakan keywords!)
5. Upload 10+ foto berkualitas
6. Verifikasi (via kartu pos atau telepon)
```

**Impact:** Muncul di Google Maps & Local Search!

#### **3. Google Analytics** (10 menit)
```
1. Buka: https://analytics.google.com
2. Create account
3. Create property: "Desa Fajar Baru"
4. Copy tracking code
5. Email tracking code ke developer untuk dipasang di website
```

**Impact:** Track visitors, analyze traffic!

## ‚ùì FAQ

**Q: Berapa lama sampai ranking #1?**
A: Untuk brand keywords ("desa fajar baru"): 1-3 bulan. Untuk generic keywords ("smart village"): 6-12 bulan.

**Q: Apakah perlu bayar Google Ads?**
A: Tidak wajib untuk SEO organik. Tapi bisa membantu untuk awareness awal.

**Q: Berapa biaya SEO ini?**
A: Setup awal: FREE (semua tools Google gratis). Maintenance: waktu Anda untuk update konten.

**Q: Apakah perlu hire SEO agency?**
A: Tidak wajib. Guide ini cukup untuk ranking lokal. Agency perlu jika target nasional.

**Q: Apa yang paling penting?**
A: 3 hal: Google Search Console, Google Business Profile, dan konten berkualitas.
